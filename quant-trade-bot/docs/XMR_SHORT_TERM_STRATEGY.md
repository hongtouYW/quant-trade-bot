## 🔥 短线策略推荐（1-3天）

### 策略1: 布林带 + RSI 反转策略 ⭐⭐⭐⭐⭐
**最适合XMR短线**

#### 指标配置
```python
# 布林带 (Bollinger Bands)
BB_PERIOD = 20
BB_STD = 2.0  # 标准差

# RSI (相对强弱指数)
RSI_PERIOD = 14
RSI_OVERSOLD = 30  # 超卖线
RSI_OVERBOUGHT = 70  # 超买线
```

#### 买入信号（做多）
1. ✅ 价格触及或突破布林带下轨
2. ✅ RSI < 30（超卖）
3. ✅ 成交量放大（> 平均1.5倍）
4. ✅ 出现阳线反弹

#### 卖出信号（平仓）
1. 🎯 价格触及布林带上轨
2. 📈 RSI > 70（超买）
3. 🛡️ 止损: -2%
4. 💰 止盈: +2%

#### 优势
- ⚡ 反应快，适合捕捉短期反弹
- 📊 胜率高（60-70%）
- 🎯 风险可控
- 💡 适合高波动币种

2. 出现长上影线（卖盘强）
3. 成交量萎缩

## 🎯 具体代码实现

### XMR短线策略示例
```python
def xmr_short_term_strategy(df):
    """XMR短线策略 - RSI + 布林带"""
    
    # 计算指标
    df['rsi'] = talib.RSI(df['close'], timeperiod=9)  # 快速RSI
    df['bb_upper'], df['bb_middle'], df['bb_lower'] = talib.BBANDS(
        df['close'], timeperiod=15, nbdevup=2.5, nbdevdn=2.5
    )
    df['volume_sma'] = talib.SMA(df['volume'], timeperiod=20)
    
    latest = df.iloc[-1]
    prev = df.iloc[-2]
    
    # 买入信号
    buy_signal = (
        latest['close'] <= latest['bb_lower'] * 1.01 and  # 触及下轨
        latest['rsi'] < 35 and  # RSI超卖
        latest['volume'] > latest['volume_sma'] * 1.5 and  # 放量
        latest['close'] > prev['close']  # 出现反弹
    )
    
    # 卖出信号
    sell_signal = (
        latest['close'] >= latest['bb_upper'] * 0.99 or  # 触及上轨
        latest['rsi'] > 70  # RSI超买
    )
    
    return {
        'action': 'buy' if buy_signal else ('sell' if sell_signal else 'hold'),
        'price': latest['close'],
        'rsi': latest['rsi'],
        'bb_position': (latest['close'] - latest['bb_lower']) / 
                       (latest['bb_upper'] - latest['bb_lower']) * 100
    }
```
