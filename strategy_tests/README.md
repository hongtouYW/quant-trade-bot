# 策略测试文件

本文件夹包含所有交易策略的测试和分析文件。

## 📂 文件类型

### 策略文件
- `*strategy*.py` - 策略实现和分析
- `*backtest*.py` - 回测程序
- `*test*.py` - 测试程序

### 数据文件
- `*analysis*.json` - 策略分析结果
- `*trading*.json` - 交易记录
- `*backtest*.json` - 回测结果
- `potential_*.json` - 潜力币扫描结果
- `big_money*.json` - 大资金追踪数据

## 📊 主要策略

- 多时间框架策略
- 简单策略回测
- 大资金追踪
- 潜力币扫描

## 💡 使用说明

这些文件主要用于历史策略测试和分析，不影响当前的XMR实盘监控。
