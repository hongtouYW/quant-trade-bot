# Flash Quant V2 — 全新量化策略研究

> **日期**: 2026-04-15
> **背景**: V1 爆发策略三版回测全亏 (胜率 6-11%), 需要重新选策略
> **原则**: 只选有回测数据支撑的策略, 不凭空设计

---

## 📊 V1 失败的教训

```
V1 的核心问题:
1. 追涨杀跌 (涨了 2% 去追 = 买在高点)
2. 止损太窄 (0.5% = 正常波动就扫)
3. 方向判断靠算法 (10% 胜率 = 比抛硬币还差)

LUKE 交易员的成功:
1. 胜率 65% (人判断方向)
2. 止损 3.7% (给足空间)
3. 盈亏比 0.99 (赢亏差不多, 靠胜率赚)
```

**结论: 不要追求高盈亏比低胜率, 要追求高胜率合理盈亏比。**

---

## 🔍 研究发现: 5 个被验证的策略方向

### 策略 1: RSI 超卖反弹 (均值回归)

**原理**: 价格跌多了会反弹, RSI < 30 时做多

**研究数据**:
- QuantifiedStrategies: RSI 策略回测 91% 胜率 (股票)
- 加密市场: RSI + 趋势过滤, 胜率 60-70%
- Stochastic RSI 回测: 78% 胜率
- **关键**: 只在上涨趋势中买超卖, 不在下跌趋势中

**适合我们的原因**:
- 高胜率 (60-70%) — 和 LUKE 接近
- 不追涨, 是在价格回调时买入
- 逻辑简单, 参数少
- 有大量公开回测数据

**风险**:
- 加密市场 RSI 单独使用效果差 (需要配合趋势)
- 真正的崩盘时超卖会更超卖

---

### 策略 2: 网格交易 (Grid Trading)

**原理**: 在价格区间内预设多个买卖点, 低买高卖

**研究数据**:
- 历史回测: 震荡市年化 15-60%
- 案例: BTC 横盘 5 个月, 网格赚 75% ROI
- BingX: 6.7 亿美金网格交易量, 16 万活跃用户
- **最适合震荡市** (我们 V1 在震荡市亏最多)

**适合我们的原因**:
- 不需要判断方向! (震荡市自动低买高卖)
- 高频小利润累积
- 数学上可证明盈利 (只要价格在区间内)

**风险**:
- 单边趋势会亏 (价格跌出区间 → 被套)
- 需要足够本金铺网格
- 手续费是大头

---

### 策略 3: 趋势跟踪 + 回踩入场

**原理**: 确认趋势后, 等回调再入场 (不追高)

**研究数据**:
- 趋势跟踪在加密市场表现最好 (大趋势明显)
- 回测: 42% 胜率但盈亏比 5.25 (平均赢 21%, 亏 4%)
- 135 笔交易, 平均每笔赚 7%

**和 V1 的区别**:
```
V1: 涨了 2% → 追进去 (买在高点)
V3: 涨了 → 等回调到 EMA → 在支撑位买 (买在回调低点)
```

**适合我们的原因**:
- 和 V1 最接近, 代码改动最小
- 不追高, 在回调时入场
- 盈亏比高

**风险**:
- 胜率只有 42% (需要高盈亏比补)
- 可能等很久没有回调

---

### 策略 4: 资金流向 + 波动率突破

**原理**: 监控大额资金流入/流出, 配合波动率收缩后突破

**研究数据**:
- 遗传算法优化: BTC +29%, ETH +550%
- 波动率收缩 (BB 收窄) 后爆发是经典形态
- 和 V1 爆发策略类似, 但入场时机不同

**区别**:
```
V1: 爆发已经发生 → 追进去
V4: 爆发前 (BB 收窄) → 提前布局 → 等爆发
```

**风险**:
- BB 收窄后可能不爆发 (继续横盘)
- 需要判断爆发方向

---

### 策略 5: 配对交易 / 统计套利

**原理**: 找两个高度相关的币 (如 BTC/ETH), 当价差偏离时做套利

**研究数据**:
- 学术论文支持 (协整检验)
- 市场中性 (不依赖方向判断)
- 稳定低收益

**风险**:
- 收益低 (年化 10-20%)
- 需要同时做多一个做空一个
- 相关性可能突然打破

---

## 🎯 我的推荐: 策略 1 + 3 组合

### V2 方案: "回踩做多" 策略

```
核心逻辑:
  1. 大趋势判断: 1H EMA20 > EMA50 → 上涨趋势 (只做多)
  2. 等待回调: 5min/15min RSI 跌到 30-40 (超卖区)
  3. 入场: 价格触碰 EMA20 或布林带下轨 → 做多
  4. 止损: 前低下方 1% (大约 2-3% 价格空间)
  5. 止盈: 前高或布林带上轨 (大约 2-3% 价格空间)
```

### 为什么选这个?

| 指标 | V1 爆发策略 | V2 回踩做多 | LUKE |
|---|---|---|---|
| 入场时机 | 涨完了追 | **回调时买** | 人判断 |
| 胜率 (预期) | 10% | **55-65%** | 65% |
| 盈亏比 | 4-6 | **1.0-1.5** | 0.99 |
| 止损空间 | 0.5% | **2-3%** | 3.7% |
| 方向判断 | 5min 涨跌 | **1H 趋势** | 人脑 |

### 和 V1 的关键区别

```
V1: "涨了就追" → 买在高点 → 回调就亏
V2: "跌了才买" → 买在低点 → 反弹就赚

V1: 0.5% 止损 → 正常波动就扫
V2: 2-3% 止损 → 需要真正跌穿支撑才亏

V1: 信号多 (日均 10+) → 大部分是噪音
V2: 信号少 (日均 3-5) → 每个都是回调机会
```
