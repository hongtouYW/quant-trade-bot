## 改动内容 (What)

### 1. 持仓表 — 从 9 列扩展到 17 列
新增字段:
| 字段 | 说明 |
|---|---|
| 持仓量 | 合约数量 (4位小数) |
| MMR | 维持保证金率 % |
| 均价 | 仓位平均价格 |
| 标记价 | Binance 标记价格 |
| 强平价 | 预估强平价格 (红色) |
| 止盈 | 止盈价格 (绿色) |
| 止损 | 止损价格 (红色) |
| 手续费 | 开仓手续费 |
| 隔夜费 | 资金费率累计 |
| 实际盈亏 | 未实现盈亏 - 总费用 |

### 2. 交易页 — 从 10 列扩展到 19 列
新增字段:
| 字段 | 说明 |
|---|---|
| 持仓量 | 4位小数 |
| 开仓费 / 平仓费 / 隔夜费 / 总费用 | 分项手续费 |
| 毛利 / 净利 | 区分费前费后 |
| 止盈 / 止损 | 每笔设定价格 |
| **点击行弹出图表** | Canvas 价格曲线 |

### 3. 新增页面: /history — 日历交易历史
- 日历网格显示当月每一天
- 绿色格 = 当日盈利, 红色格 = 当日亏损
- 显示每日: 交易笔数 + 胜负 + 净利
- 点击日期展开当日交易明细表
- 每笔可点"查看"弹出图表

### 4. 图表功能 (Canvas 绘制)
- 价格曲线 (绿盈红亏)
- 入场线 (蓝色虚线 + "买"标记)
- 平仓线 (绿/红虚线 + "卖"标记)
- 止盈线 (绿色虚线 + "止盈"标签)
- 止损线 (红色虚线 + "止损"标签)
- 渐变填充区域
- 底部信息: 入场/出场/止盈/止损/保证金/持仓时长/各项费用/毛利/净利/ROI/平仓原因

### 5. 统计卡片新增
- 总手续费 (含隔夜费分项)
- 平均持仓时长

### 6. 其他修复
- 关闭 5 秒自动刷新 (防闪烁)
- Mock 数据固定种子 (刷新不变)
- 时区改为马来西亚 UTC+8
- 中文化全部界面
- 持仓量显示 4 位小数 (.0000)
- 导航栏加"历史"入口 + 手动"刷新"按钮
