## 方案: 双时间框架确认 (MTF)

```
第一层: 1H EMA20/EMA50 判断大趋势
  EMA20 > EMA50 + 近3根2涨 → bullish → 只做多

  差距 < 0.1% 或动量不足 → sideways → 不开仓

第二层: 5min 爆发信号 (量比+涨跌)
  只做与大趋势一致的方向
```
